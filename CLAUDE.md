# Regira Source Codebase

This is the **Regira library source repository** — a collection of NuGet packages intended to be consumed by other projects.

> **Important:** The `ai/` folder at the root of this repository contains guides for **consuming projects**, not for working on this codebase. Do not follow `ai/AGENTS.md` when contributing to the source — it is a consumer bootstrap guide that gets extracted into downstream projects via NuGet.

For working on this codebase, read `AGENTS.md` (this directory) first.
