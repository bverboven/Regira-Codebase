## Quick Start Checklist

When implementing a new entity in an application:

*Required*
- [ ] Create entity model(s) with appropriate interfaces
- [ ] Configure DbContext
- [ ] Create controller

*Recommended*

- [ ] Create DTOs (output DTO, input DTO)
- [ ] Configure mapping

*Optional*

- [ ] Create SearchObject
- [ ] Implement query filters
- [ ] Add Processors
- [ ] Add Preppers
- [ ] Add Primers
- [ ] Configure child properties with Related method
- [ ] Register entity services in DI
