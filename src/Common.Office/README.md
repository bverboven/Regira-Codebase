# Regira Office

Regira Office is a collection of document and communication libraries for .NET applications. All modules share a common set of abstractions from `Common.Office` and are designed to be interchangeable.

## Modules

| Module | Description |
|--------|-------------|
| [Barcodes](docs/barcodes/README.md) | Barcode and QR code generation and scanning |
| [CSV](docs/csv/README.md) | CSV reading and writing via CsvHelper |
| [Excel](docs/excel/README.md) | Excel workbook reading and writing |
| [Mail](docs/mail/README.md) | Email sending via SendGrid and Mailgun |
| [OCR](docs/ocr/README.md) | Optical character recognition via Tesseract and PaddleOCR |
| [PDF](docs/pdf/README.md) | HTML→PDF, PDF operations, and printing |
| [VCards](docs/vcards/README.md) | vCard contact file reading and writing |
| [Word](docs/word/README.md) | Word document creation, conversion, merge, and extraction |
