# Regira Office

Regira Office is a collection of document and communication libraries for .NET applications. All modules share a common set of abstractions from `Common.Office` and are designed to be interchangeable.

## Modules

| Module | Description |
|--------|-------------|
| [Barcodes](docs/barcodes) | Barcode and QR code generation and scanning |
| [CSV](docs/csv) | CSV reading and writing via CsvHelper |
| [Excel](docs/excel) | Excel workbook reading and writing |
| [Mail](docs/mail) | Email sending via SendGrid and Mailgun |
| [OCR](docs/ocr) | Optical character recognition via Tesseract and PaddleOCR |
| [PDF](docs/pdf) | HTML→PDF, PDF operations, and printing |
| [VCards](docs/vcards) | vCard contact file reading and writing |
| [Word](docs/word) | Word document creation, conversion, merge, and extraction |
